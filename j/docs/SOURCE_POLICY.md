# 情報源ポリシー

## 対象

このデータベースは漫画『呪術廻戦』のみを対象とします。

- 0巻全4話
- 本編第1話～第271話
- 単行本の描き下ろし、EXTRA PAGES、巻末補足
- 公式ファンブックのうち、原作漫画と設定に関する部分

テレビアニメ、劇場版、ゲーム、小説、舞台、『呪術廻戦≡（モジュロ）』は対象外です。

## 情報源の優先順位

1. 漫画0巻・本編単行本の該当ページ
2. 単行本の描き下ろし、EXTRA PAGES、巻末解説
3. 集英社・少年ジャンプの公式書誌、目次、原作設定
4. 公式ファンブックの原作漫画・設定部分
5. 日本語の原作漫画解説記事
6. 海外Wiki・Fandomなどの二次資料

5と6は、原作内の確認箇所を探すための補助線として扱います。二次資料だけで得た情報を、原作確認済みの確定情報とは扱いません。

## 二次資料の扱い

- 日本語の原作漫画記事を優先します。
- 海外Wiki・Fandomは、国内資料で不足する人物、場所、技、登場話を探す索引に限定します。
- アニメ情報と漫画情報が同じページに混在する場合、漫画として明示された箇所だけを候補にします。
- 二次資料の文章は転載せず、複数資料を比較して独自の日本語要約を作ります。
- 資料間で食い違う場合は、未確定のまま確認キューへ残します。

## 使用禁止

- テレビアニメ固有の描写、設定、各話構成
- 劇場版で追加・変更された描写
- ゲーム、小説、舞台の独自設定
- 声優、キャスト情報
- 『呪術廻戦≡（モジュロ）』の設定
- 出典不明のSNS投稿
- 考察、推測、通説を確定事項として記録すること
- 正式名称が不明な技へ、英訳名やファン名称を正式名称として付けること

## 確度ラベル

- `official-metadata`：話名、巻、開始ページなどを公式書誌で確認
- `first-pass-crosscheck-pending`：第一稿。個別の最終照合前
- `phaseN-crosschecked-secondary-not-primary-pages`：公式書誌と漫画限定二次資料を突合したが、単行本の該当ページを直接確認していない
- `primary-checked`：原作単行本の該当ページを直接確認
- `reviewed`：原作確認後、別工程でもう一度確認

確度を上げる場合も、旧候補や修正理由は監査記録へ残します。

## 現在の注意

現在のサイトは全範囲を閲覧できる第一稿です。話名・巻・開始ページなどの公式書誌情報は照合済みですが、全275話の単行本ページ直接監査は未着手です。画面上の「状態」と `reports/DATA_QUALITY_AUDIT.md` を参照してください。
