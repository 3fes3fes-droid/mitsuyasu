# 情報源ポリシー

## 最優先

1. 漫画0巻・本編単行本
2. 漫画単行本の描き下ろし、EXTRA PAGES、巻末解説
3. 集英社・少年ジャンプの公式書誌、目次、原作設定
4. 公式ファンブックの原作漫画・設定部分

## 補助資料

日本語の原作漫画記事を優先する。英語Fandomは、国内資料で不足する話の人物・場所・技を探す索引としてのみ利用する。

補助資料から得た情報は、原作本文で直接確認するまでは `secondary` または `crosschecked-secondary` とする。

## Phase 1で使用した補助線

- 0巻：集英社公式書誌＋Fandomの漫画章ページ
- 本編第1～18話：集英社公式書誌＋マンガゲームの原作漫画巻別ページ
- 過去の `ju.txt`：候補の比較と欠落検出だけに使用

補助資料の文を転載せず、複数項目を比較して独自の日本語要約を作成する。

## 使用禁止

- テレビアニメの各話ページ、描写、設定
- 劇場版で追加・変更された描写
- ゲーム独自設定
- 小説独自設定
- 舞台独自設定
- 声優・キャスト情報
- 『呪術廻戦≡（モジュロ）』
- 出典不明のSNS投稿
- 考察を確定事項として扱うこと

同じサイト内に漫画記事とアニメ記事が混在する場合、URL単位で漫画記事だけを登録する。

## 確度ラベル

- `official-metadata`：話名、巻、開始ページなど公式書誌で確認
- `first-pass-crosscheck-pending`：第一稿。最終照合前
- `phase1-crosschecked-secondary-not-primary-pages`：公式書誌と漫画限定二次資料を突合。原作ページ直接確認前
- `primary-checked`：原作単行本ページを直接確認
- `reviewed`：別工程でもう一度確認

確度を上げる際は、古い候補を削除せず監査記録へ残す。
