# 呪術廻戦 原作漫画データベース

漫画0巻全4話、本編第1話～第271話、30巻描き下ろしエピローグを対象としたPC向け静的HTMLデータベースです。アニメ、劇場版、ゲーム、小説、舞台は対象外です。

## 現在の状態

この版は、**全範囲を閲覧できる動作する第一稿**です。

- 全275話の短いあらすじ・詳細あらすじを収録
- 人物189件、技・術式179件、用語173件を収録
- 各話・人物・技・用語・補遺・出典の画面を実装
- 全816件の詳細表示を自動巡回済み
- 単行本全ページを直接開いた原典監査は未完了
- 人物178件、技・術式121件、用語84件は個別精査待ち

画面右上の「状態」で、現在の完成段階を確認できます。

## 開き方

通常は最上位の `index.html` をChromeで開きます。

ローカルファイルのscript読込をブラウザ設定が遮断する場合は、フォルダ内で次を実行します。

```powershell
python tools/serve.py
```

表示された `http://127.0.0.1:8000/index.html` を開きます。

## 編集手順

1. `data/source` の原本データを修正する
2. `python tools/build_site.py` を実行する
3. `python tools/validate_site.py` を実行する
4. `python tools/audit_data_quality.py` を実行する
5. HTMLを開いて確認する
6. `python tools/make_manifest.py` を実行する
7. Gitへcommitする

`data/generated` は表示用の生成物です。直接編集しません。

## 軽量化

旧版は各話ページで約1MBのデータを初期読込していました。修正版は一覧データと選択中チャンクだけを読み、各話ページの初期読込量を約77.6%削減しています。

詳細は `reports/PERFORMANCE_AUDIT.md` を参照してください。
